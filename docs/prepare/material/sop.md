# Personal Statement/Statement of Purpose 个人陈述/目的陈述

PS/SOP是文书的重要环节，是学校的命题作文，学校项目会给出内容的要求。

PS通常包括自己的家庭情况、爱好特长，向committee展示自己的特质。

SOP侧重purpose，包括选择这个专业/项目的原因，过往的相关经历，从经历中获得了什么，感兴趣的方向，未来发展规划。
